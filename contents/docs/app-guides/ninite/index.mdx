> Install and Update All Your Programs at Once
No toolbars. No clicking next. Just pick your apps and go.

## Installation

<Stepper>
  <StepperItem title="Step 1: Get the installer">
    The ninite installer can be found on
    [Ninite.com](https://ninite.com/).
  </StepperItem>
  <StepperItem title="Step 2: Select your programs">
    The [ninite](https://ninite.com/) website provides a list of programs available for installation, select the checkboxes for desired programs and download the installer.
  </StepperItem>
  <StepperItem title="Step 3: Install using ninite">
    Simply run the downloaded ninite.exe file and it will automatically install the selected apps, if already installed it will skip over or update the application. You can keep the installer and run it occasionally to update all the selected programs.
  </StepperItem>
</Stepper>

### Features

- No toolbars or extra junk
- Won't bother you with any choices or options
- Install apps in their default location
- Download apps from each publisher's official site

### Usage

The ninite installer is a no config, no options needed installer which will automatically install all selected applications in the background.

Simply run the downloaded executable to install the programs.
